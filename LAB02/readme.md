Программа не запустилась, потому что невозможно запустить исходный код программы напрямую. Сначала должна произойти компиляция, которую флаг --no-build предотвращает.
Когда запуск программы совершается командой dotnet run без флага --no-build, программа не скомпилируется заранее.
Запускается выполнимый файл (exe-файл), который получается в ходе компилирования. Без выполнимого файла нечего будет и запускать.
