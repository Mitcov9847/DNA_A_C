# Лабораторная работа №5
**Тема:** Что такое функции, вызов функций, stack

## Концепты

1. **Выполнимый файл (executable)** — машинный код, готовый к выполнению процессором.
2. **Функция** — блок кода, выполняющий конкретную задачу, может быть вызван из других частей программы.
3. **Instruction Pointer (IP)** — адрес текущей выполняемой инструкции.
4. **Stack и Stack Pointer (SP)** — структура данных для хранения локальных переменных, адресов возврата и аргументов функций.
5. **Адрес возврата** — адрес инструкции, на которую функция возвращается после завершения.
6. **Локальная переменная** — переменная, существующая только в рамках функции, хранится на стеке.
