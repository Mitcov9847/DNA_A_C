**Тема:** Что такое функции, вызов функций, stack

## Концепты

** Выполнимый файл — набор машинных команд, которые процессор может запускать напрямую.
** Функция — отдельная часть программы, выполняющая конкретную задачу и вызываемая из других участков кода.
** Instruction Pointer (IP) — регистр, в котором хранится адрес текущей выполняемой инструкции.
** Стек и Stack Pointer (SP) — стек используется для хранения временных данных: аргументов, локальных переменных и адресов возврата. SP указывает на его текущую вершину.
### Адрес возврата — точка в программе, куда управление возвращается после выполнения функции.
### Локальная переменная — переменная, существующая только внутри функции и размещённая обычно в стеке.
